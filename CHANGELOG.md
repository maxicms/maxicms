# Changelog

All notable changes to MaxiCMS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-27

### Added
- Initial release as independent MaxiCMS fork
- PHP 8.0+ support (raised from PHP 7.2)
- composer.json with dependencies
- GitHub Actions CI workflow

### Changed
- Rebranded from InstantCMS to MaxiCMS
- Updated version to 0.1.0
- All SQL packages author changed from "InstantCMS Team" to "MaxiCMS Team"
- All URLs updated from instantcms.ru to maxicms.ru
- SVG logos aria-label updated to MaxiCMS

### Removed
- Update directory (not needed for independent fork)
- Historical InstantCMS update packages

### Fixed
- All remaining InstantCMS references cleaned from codebase
